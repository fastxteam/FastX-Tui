# FastX-Tui 文档

欢迎使用 FastX-Tui 文档！

## 什么是 FastX-Tui？

FastX-Tui 是一个统一视图架构的终端用户界面应用程序，提供了强大的功能和灵活的插件系统。

## 主要功能

- **统一视图架构**：提供一致的用户体验
- **插件系统**：支持动态加载和扩展功能
- **异步任务管理**：非阻塞的命令执行机制
- **实时日志管理**：全面的日志查看和分析功能
- **配置管理**：灵活的配置选项和设置界面
- **多语言支持**：支持多种语言界面
