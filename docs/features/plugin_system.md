# 插件系统

## 概述

FastX-Tui 插件系统是一个强大的扩展机制，允许开发者和用户通过插件扩展 FastX-Tui 的功能。插件系统支持多文件结构、二进制文件和在线安装，为开发者提供了极大的灵活性。

## 插件功能

### 1. 多文件插件结构

插件可以包含多个文件，包括：
- 入口文件（必须）
- 业务逻辑文件
- 资源文件
- 二进制文件

### 2. 插件元数据

每个插件都包含元数据，包括：
- 插件名称和版本
- 作者信息
- 描述和分类
- 依赖项
- 兼容性信息

### 3. 插件生命周期

插件有明确的生命周期：
- **加载**：插件被加载到内存中
- **初始化**：插件资源被初始化
- **注册**：插件向系统注册命令和菜单
- **运行**：插件执行其功能
- **清理**：插件资源被清理
- **卸载**：插件被卸载

### 4. 插件通信

插件可以：
- 调用系统 API
- 访问系统资源
- 与其他插件通信
- 向系统注册事件处理器

## 插件管理

### 1. 查看已安装的插件

1. 从主菜单选择「插件管理」
2. 选择「已安装插件」查看所有已安装的插件

### 2. 启用/禁用插件

1. 从主菜单选择「插件管理」
2. 选择「已安装插件」
3. 使用方向键选择插件
4. 按 `Space` 键启用或禁用插件

### 3. 安装新插件

FastX-Tui 支持多种安装插件的方式：

#### 本地安装

1. 从主菜单选择「插件管理」
2. 选择「安装插件」
3. 选择「本地安装」
4. 输入插件目录路径
5. 按 `Enter` 键安装

#### 在线安装

1. 从主菜单选择「插件管理」
2. 选择「安装插件」
3. 选择「在线安装」
4. 输入插件 GitHub 仓库 URL
5. 按 `Enter` 键安装

### 4. 更新插件

1. 从主菜单选择「插件管理」
2. 选择「已安装插件」
3. 使用方向键选择需要更新的插件
4. 按 `U` 键更新插件

### 5. 删除插件

1. 从主菜单选择「插件管理」
2. 选择「已安装插件」
3. 使用方向键选择需要删除的插件
4. 按 `Delete` 键删除插件
