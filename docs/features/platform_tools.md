# 平台工具

## 概述

FastX-Tui 提供了一系列内置的平台工具，用于管理和监控系统资源，执行常见操作。

## 系统监控

FastX-Tui 可以实时监控系统资源使用情况，包括：

- CPU 使用率
- 内存使用情况
- 磁盘空间
- 网络连接

### 查看系统信息

1. 从主菜单选择「系统监控」
2. 选择「系统信息」查看详细的系统信息

### 资源监控

1. 从主菜单选择「系统监控」
2. 选择「资源监控」查看实时的资源使用情况

## 文件管理

FastX-Tui 提供了基本的文件管理功能：

- 浏览文件系统
- 创建、删除、重命名文件和目录
- 查看文件内容
- 搜索文件

### 使用文件管理器

1. 从主菜单选择「文件管理」
2. 使用方向键浏览文件系统
3. 按 `Enter` 键打开文件或进入目录
4. 按 `F2` 键重命名文件或目录
5. 按 `F3` 键查看文件内容
6. 按 `Delete` 键删除文件或目录

## 进程管理

FastX-Tui 可以管理系统进程：

- 查看运行中的进程
- 终止进程
- 查看进程详细信息

### 管理进程

1. 从主菜单选择「进程管理」
2. 使用方向键选择进程
3. 按 `Enter` 键查看进程详细信息
4. 按 `Delete` 键终止进程

## 网络工具

FastX-Tui 提供了一些网络工具：

- 网络连接查看
- 端口扫描
- Ping 工具
- Traceroute 工具

### 使用网络工具

1. 从主菜单选择「网络工具」
2. 选择您需要使用的网络工具
3. 按照提示输入相关参数
4. 查看结果

## 终端工具

FastX-Tui 内置了终端模拟器，您可以在 FastX-Tui 中直接执行命令：

1. 从主菜单选择「终端工具」
2. 在终端中输入命令
3. 按 `Enter` 键执行命令

## 快捷键

| 快捷键 | 功能 |
|-------|------|
| `F2` | 重命名文件/目录 |
| `F3` | 查看文件内容 |
| `Delete` | 删除文件/目录/终止进程 |
| `Ctrl+C` | 中断当前操作 |
