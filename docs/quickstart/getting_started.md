# 入门指南

## 启动 FastX-Tui

安装完成后，可以使用以下命令启动 FastX-Tui：

```bash
fastx-tui
```

## 基本界面

FastX-Tui 启动后，您会看到以下界面：

### 1. 顶部菜单

顶部显示当前视图的标题和面包屑导航，显示当前所在的视图层级。

### 2. 主内容区

主内容区显示当前视图的详细内容，例如菜单列表、日志信息等。

### 3. 底部状态栏

底部状态栏包含三个部分：

- **左侧**：显示当前系统状态信息
- **中间**：显示当前视图的状态
- **右侧**：显示快捷键提示

## 基本操作

### 1. 导航菜单

使用键盘方向键（上/下）选择菜单选项，按 `Enter` 键进入子菜单或执行命令。

### 2. 返回上一级

按 `Esc` 键或 `Backspace` 键返回上一级菜单。

### 3. 查看任务列表

按 `T` 键查看正在运行和已完成的任务列表。

### 4. 打开设置

按 `S` 键打开设置界面，您可以在这里配置 FastX-Tui 的各种选项。

### 5. 查看帮助

按 `H` 键查看帮助信息，了解 FastX-Tui 的功能和使用方法。

### 6. 退出应用

按 `Q` 键退出 FastX-Tui 应用。

## 常用功能

### 1. 插件管理

FastX-Tui 支持插件扩展，您可以：

- 查看已安装的插件
- 启用/禁用插件
- 安装新插件
- 更新插件

### 2. 日志管理

FastX-Tui 提供了强大的日志管理功能：

- 实时查看日志
- 按级别筛选日志
- 搜索日志内容
- 导出日志文件

### 3. 配置管理

您可以通过设置界面配置 FastX-Tui 的各种选项：

- 外观设置
- 行为设置
- 插件设置
- 高级设置

### 4. 任务管理

FastX-Tui 使用异步任务队列处理命令执行：

- 非阻塞执行命令
- 查看任务状态和结果
- 取消正在执行的任务
