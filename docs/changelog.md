## 版本号规则

FastX-Tui 使用语义化版本号（Semantic Versioning）规则：

- **主版本号（Major）**：当进行不兼容的 API 更改时递增
- **次版本号（Minor）**：当添加功能且保持向后兼容时递增
- **修订号（Patch）**：当进行向后兼容的错误修复时递增

## 升级指南

### 从旧版本升级

```bash
# 使用 pip 升级
pip install --upgrade fastx-tui

# 使用 uv 升级
uv pip install --upgrade fastx-tui
```

### 升级注意事项

1. **备份配置文件**：在升级前建议备份配置文件
2. **检查插件兼容性**：确保使用的插件与新版本兼容
3. **查看更新日志**：了解新版本的变化
4. **测试功能**：升级后测试关键功能是否正常工作
